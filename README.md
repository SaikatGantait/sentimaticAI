# Crypto Sentiment Trader

AI-assisted crypto sentiment dashboard with token insights, wallet tracking, influencer monitoring, and simulated auto-trading. Built with Vite, React, TypeScript, Tailwind, and shadcn-ui.

## Getting Started

Requirements: Node.js 18+

```sh
npm i
npm run dev
```

## Features

- Token list with search, filter, and sorting
- Token details with sentiment, liquidity, and recommendations
- Wallet tracking with add/remove and local persistence
- Influencer tracker
- Quick trade simulator and auto-trade panel
- Light/Dark theme toggle

## Scripts

- `npm run dev`: Start dev server
- `npm run build`: Build for production
- `npm run preview`: Preview production build
